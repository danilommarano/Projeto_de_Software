## 1. Analise o cenário de negócio do tema de projeto escolhido pelo seu grupo

O projeto Livre visa construir uma plataforma digital que possa conectar bibliotecas e leitores do Brasil inteiro. Em 2021 as bibliotecas precisam se conectar com seus leitores através da internet para que possam se inserir no movimento tecnológico imposto sobre o mercado literário. Com isso a Livre pretende oferecer aos seus usuários os seguintes recursos:

1. Oferecer um meio de comunicação entre o leitor e o vendedor atendendo suas nescessidades, seja na customização e detalhamento da elaboração de um anúncio ou um chat em que possam ser esclarecidas dúvidas sobre o produto.
  
2. Produzir mecanismos de barateamento dos livros para que possamos atender públicos de baixa renda que geralmente não são inclusos neste mercado. Alguns dos mecanismos seria: 

    - Promoções baseadas na popularidade.

    - Sorteios para os usuários da plataforma

    - Doações para entidades parceiras

3. Acompanhamento do status do produto:

    - Pendente
    - Aprovado
    - Faturado
    - Enviado
    - Finalizado
    - Cancelado


## 2. Definir as fronteiras do negócio a ser modelado. Definir quem e o quê pode ter interação com o negócio

1. Clientes
  Pessoas físicas ou jurídicas que desejam comprar livro. 

2. Vendedores
  Livrarias (pequenas ou grandes), editoras e sebos.
  
3. Fornecedores 
  Empresas de logística que fazem entregas. Os dados do nosso sistema irão alimentar os sistemas das empresas de logística parceiras.


## 3. Liste as atividades de negócio. Faça uma lista das atividades de negócio que cada elemento participa.

1. Clientes
  Cadastrar a conta de usuário cliente, fazer um pedido, pagar o livro, receber em casa, avaliar os livros.

2. Vendedores
  Cadastrar a conta de usuário vendedor, cadastrar os livros, receber pedidos, enviar o pedido pro fornecedor.
  
3. Fornecedores 
  Receber os pedidos, mostrar status do pedido, buscar os livros no local marcado pelos vendedores, enviar os pedidos até o cliente, confirmar que o pedido foi enviado.
